Install:

1) Install node.js - Download

2) Install the Angular CLI

npm install -g @angular/cli

3) Access a folder with the application files

4) Install the application

npm install

5) Run the application

ng serve --open
Division of folders:

directives - global directives

pipes - global pipes

services - global services

pages - directory with the pages of your application

shared - directory with shared components

shared/ui-elements - ui components

main - components of the panel structure

main/main-header - general header component

main/main-footer - general footer component

main/main-nav - general nav lateral component

main/modal-settings - modal settings component

Sass variables:

Control theme colors and other configuration variables in sass/bootstrap/_variables.scss
